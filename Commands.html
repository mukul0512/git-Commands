Some important git commands

  • git clone- is used to target an existing repository and create a clone, or copy of the target repository.
  • git status- displays the state of the working directory and the staging area.
  • git add- adds a change in the working directory to the staging area.
  • git commit- captures a snapshot of the project's currently staged changes.
  • git push- to upload local repository content to a remote repository.

Some important terminal commands

  • pwd- stands for Print Working Directory. It prints the path of the working directory, starting from the root.
  • ls- to list computer files in Unix and Unix-like operating systems. It lists the files in the current working directory.
  • cd- also known as chdir (change directory), is a command-line shell command used to change the current working directory in various operating systems.
  • clear- is used to bring the command line on top of the computer terminal.
